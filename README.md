# AI Tool Guessing Game 🤖

AI Tool Guessing Game is a simple C-based command-line game where the player tries to guess the name of a popular AI tool based on hints provided by the program.

The game gives clues about different AI tools and the player must type the correct name. If the answer is correct, the player earns points; otherwise the program shows the correct answer. This project demonstrates basic programming concepts such as conditional statements, loops, string comparison, and user input handling in C.

## Features

* Guess popular AI tool names from hints
* Simple command-line interface
* Score tracking system
* Beginner-friendly C programming project
